# Cognaize Website

Website for Cognaize Systems — cognaizesys.com.
Tech stack: Next.js 16, React 19, Tailwind CSS, Framer Motion, Lucide icons.

## Products

Six products with "[Domain] Intelligence" naming:
- Design Intelligence (drawing review)
- Manufacturing Intelligence (feasibility, routing, costing)
- Planning Intelligence (production scheduling)
- Execution Intelligence (shop floor tracking)
- Downtime Intelligence (line stoppage tracking)
- Quality Intelligence (digital inspection)
